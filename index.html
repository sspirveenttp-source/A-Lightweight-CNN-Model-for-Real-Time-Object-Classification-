Project Overview

Abstract

There has been a significant rise in the use of autonomous unmanned aerial vehicles (UAVs) for surveillance, agriculture, traffic analysis, and public safety. Real-time object classification from UAV video is an essential capability for building intelligent and automated monitoring systems. However, many deep-learning models used today are large, slow, and require expensive hardware, making them unsuitable for real-time deployment on low-cost UAVs.

To overcome this problem, this project proposes a lightweight and completely free AI-based CNN model using depthwise separable convolutions for fast and accurate object classification from UAV video streams. The proposed CNN architecture contains fewer than 2 million parameters, making it highly efficient for embedded devices.
